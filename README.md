🛒 Grocery App (Blinkit UI Clone)

<img src="grocery2.png" width="300" height="600" />
A modern grocery delivery app inspired by Blinkit, designed for a fast, smooth, and user-friendly shopping experience.

🚀 Features
🏠 Attractive Home UI with offers
🔍 Product search functionality
🛒 Add to cart system
⚡ Fast delivery concept (16 mins UI)
🎉 Promotional banners & deals
📱 Responsive design
